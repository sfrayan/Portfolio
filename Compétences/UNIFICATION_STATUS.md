## Unification Template

All skill pages follow this **unified structure**:

### Structure
```
1. Header
   - Emoji Logo (80px)
   - Technology Name (H1 with gradient)
   - Short Description (Subtitle)

2. Description Section
   - Detailed overview of the technology
   - Key capabilities and use cases

3. Associated Skills Section
   - 6 core skills related to the tech
   - Grid layout with hover effects

4. Projects Section
   - 3 representative projects
   - Card-based layout with hover animations

5. Mastery Section
   - Advanced proficiency statement
   - Key achievements and applications
```

### Design System

**Colors**: Each tech has a unique gradient
- Primary color: Specific to the technology
- Accent color: Derived from primary
- Dark background: `#1f2937` (consistent across all)
- Glassmorphism: `rgba(255, 255, 255, 0.05)` + `blur(10px)`

**Animations**:
- `fadeInDown` - Header slides in
- `bounceIn` - Logo bounce effect
- `fadeInUp` - Sections slide up
- `Hover effects` - Cards lift and glow

**Responsive**:
- Desktop: Full grid layout
- Tablet: 2-column grid
- Mobile: 1-column stacked layout

FreeRADIUS


Description

FreeRADIUS est un serveur RADIUS open-source haute performance utilisé pour l'authentification, l'autorisé et la comptabilisation (AAA) dans les réseaux informatiques. RADIUS (Remote Authentication Dial-In User Service) est un protocole réseau standard pour centraliser l'authentification des utilisateurs. FreeRADIUS supporte une variété de méthodes d'authentification : PAP, CHAP, MS-CHAP, EAP (y compris PEAP et TLS), et OAuth2. Il s'intégre avec de nombreuses sources d'authentification : LDAP, Active Directory, MySQL, PostgreSQL, et fichiers locaux. FreeRADIUS est particulièrement utilisé pour WPA-Enterprise, les VPN, les hotspots WiFi publics et l'accès à distance sécurisé. Sa scalabilité permet de gérer des milliers d'utilisateurs simultanément avec une latence minimale. Compatible avec la plupart des points d'accès WiFi et routeurs professionnels, FreeRADIUS est devenu le serveur RADIUS de référence pour les infrastructures d'authentification centralisée.

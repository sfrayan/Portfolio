Aperçu du Projet

Azure AD (Azure Active Directory) est un service d'identité et d'accès dans le cloud fourni par Microsoft. Ce projet explore l'implémentation, la configuration et la gestion centralisée des accès utilisateurs dans un environnement d'entreprise.

Ce TP couvre l'installation et la configuration d'un service d'annuaire d'entreprise, la gestion des identités, les politiques d'accès, et l'intégration avec les services Azure.

Objectifs d'Apprentissage

IAM Fondamentaux

Comprendre l'Identity & Access Management dans le cloud

Gestion des Utilisateurs

Création, gestion et synchronisation des comptes utilisateurs

Authentification

Stratégies d'authentification multi-facteurs (MFA)

Structure organisationnelle dans le cloud

Surveillance

Audit et monitoring des accès

Concepts Clés Couverts

  • Azure AD Tenant: Organisation centralisée des identités cloud
  • Applications Enregistrées: Gestion des accès d'applications tierces
  • Synchronisation Directory: Liaison entre AD on-premise et cloud
  • MFA (Multi-Factor Authentication): Sécurité avancée
  • Conditional Access: Politiques d'accès intelligentes
  • Role-Based Access Control (RBAC): Gestion granulaire des permissions
  • Application Proxy: Accès sécurisé aux apps on-premise
  • Security & Compliance: Auditing et governance

Contexte d'Entreprise

Azure AD est utilisé par des milliers d'organisations mondiales pour:

  • Gérer les accès des collaborateurs (SSO - Single Sign-On)
  • Protéger les ressources sensibles avec MFA
  • Contrôler l'accès basé sur les rôles (RBAC)
  • Intégrer des applications tierces en sécurité
  • Auditer et surveiller tous les accès
  • Se conformer aux réglementations (RGPD, HIPAA, SOC2)
